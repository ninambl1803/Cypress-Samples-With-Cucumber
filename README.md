## Getting Started

First, install packages:

```bash
npm install
```

To open Cypress

```bash
npm run cypress:open
```

If you want to run cypress

```bash
npm run cypress:run
```

If you want to run cypress and report

```bash
npm run cypress:run-n-report
```
